ODAG Wizard

    Please select your selection app from the dropdown list below

    Within this wizard you will be able to create the ODAG Load Script. The explanation below will guide you through this process

    • 1: Create selection app (this should be done before you use this wizard)
    • 2: Select the selection app created in step 1 in the dropdown below
    • 3: Fill out the fields and data source type
    • 4: Click on the Create Template App button to create your ODAG Load Script app
    • 5: Open the newly created app and edit the script accordingly (there will be detailed instructions in the script)
